# Glossario

Termos usados na plataforma Claraval e na documentacao.

## Metodologia

| Termo | Significado |
|-------|------------|
| **Pilar** | Um dos 5 eixos da metodologia: Estrategia, Governanca, Metodo CLA, Pessoas, Resultados |
| **Eixo** | Subdivisao de um pilar (ex: "Planejamento Estrategico" dentro de "Estrategia") |
| **Pergunta** | Item do questionario de diagnostico, vinculada a um eixo |
| **Nivel** | Resposta de 1-5 para cada pergunta do diagnostico (1 = imatura, 5 = excelencia) |
| **Metodo CLA** | Pilar que cobre Branding/Atracao, Comercial e Sucesso do Cliente |
| **BSC** | Balanced Scorecard — mapa estrategico da escola |
| **KPI** | Key Performance Indicator — indicador de desempenho |
| **OKR** | Objectives and Key Results — metodologia de metas |

## Contratos

| Termo | Significado |
|-------|------------|
| **Start** | Contrato de entrada — 3 meses, 1 pilar apenas |
| **Club** | Contrato de 12 meses, acesso total aos 5 pilares |
| **Corp** | Contrato de 12 meses, Club + consultoria hands-on em setor especifico |
| **Cluster** | Agrupamento de escolas por tamanho (ate 300, 300-500, 500-1000, 1000+ alunos) |

## Plataforma

| Termo | Significado |
|-------|------------|
| **Diagnostico** | Questionario de maturidade gerencial com notas por pilar/eixo |
| **Plano de Acao** | Lista de itens gerados automaticamente a partir das lacunas do diagnostico |
| **Template** | Tipo de ferramenta — formulario estruturado para documentar informacoes |
| **Score** | Tipo de ferramenta — auto-avaliacao com calculo de nota |
| **Checklist** | Tipo de ferramenta — lista de itens com checkbox e progresso |
| **Trilha** | Agrupamento de videos no e-learning |
| **Sprint** | Ciclo de trabalho (semanal ou quinzenal) para acompanhamento de tarefas |
| **Poker Points** | Estimativa de esforco para tarefas (planning poker) |
| **Flag Pirata** | Marcacao de tarefa nao planejada que entrou na sprint |
| **targetAudience** | Campo que define para quais contratos uma pergunta aparece (CLUBE, CORP, START, AMBOS) |
| **selectedPillarId** | UUID do pilar escolhido por escolas Start (restringe acesso) |

## Tecnico

| Termo | Significado |
|-------|------------|
| **NSSM** | Non-Sucking Service Manager — gerenciador de servicos Windows usado em producao |
| **MediaMTX** | Servidor de streaming open-source que converte RTMP para HLS |
| **HLS** | HTTP Live Streaming — protocolo de streaming de video |
| **RTMP** | Real-Time Messaging Protocol — protocolo usado pelo OBS para transmitir |
| **Flyway** | Ferramenta de migracao de banco de dados SQL |
| **EntityHelper** | Utilitario obrigatorio do backend para lookups de entidades |
| **Toggleable** | Interface obrigatoria para entidades com campo `active` |
| **useAdminResource** | Hook generico do frontend para CRUD admin |
