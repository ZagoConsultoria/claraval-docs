# Glossário

Termos usados na plataforma Claraval e na documentação.

## Metodologia

| Termo | Significado |
|-------|------------|
| **Pilar** | Um dos 5 eixos da metodologia: Estratégia, Governança, Método CLA, Pessoas, Resultados |
| **Eixo** | Subdivisão de um pilar (ex: "Planejamento Estratégico" dentro de "Estratégia") |
| **Pergunta** | Item do questionário de diagnóstico, vinculada a um eixo |
| **Nível** | Resposta de 1-5 para cada pergunta do diagnóstico (1 = imatura, 5 = excelência) |
| **Método CLA** | Pilar que cobre Branding/Atração, Comercial e Sucesso do Cliente |
| **BSC** | Balanced Scorecard — mapa estratégico da escola |
| **KPI** | Key Performance Indicator — indicador de desempenho |
| **OKR** | Objectives and Key Results — metodologia de metas |

## Contratos

| Termo | Significado |
|-------|------------|
| **Start** | Contrato de entrada — 3 meses, 1 pilar apenas |
| **Club** | Contrato de 12 meses, acesso total aos 5 pilares |
| **Corp** | Contrato de 12 meses, Club + consultoria hands-on em setor específico |
| **Cluster** | Agrupamento de escolas por tamanho (até 300, 300-500, 500-1000, 1000+ alunos) |

## Plataforma

| Termo | Significado |
|-------|------------|
| **Diagnóstico** | Questionário de maturidade gerencial com notas por pilar/eixo |
| **Plano de Ação** | Lista de itens gerados automaticamente a partir das lacunas do diagnóstico |
| **Template** | Tipo de ferramenta — formulário estruturado para documentar informações |
| **Score** | Tipo de ferramenta — auto-avaliação com cálculo de nota |
| **Checklist** | Tipo de ferramenta — lista de itens com checkbox e progresso |
| **Trilha** | Agrupamento de vídeos no e-learning |
| **Sprint** | Ciclo de trabalho (semanal ou quinzenal) para acompanhamento de tarefas |
| **Poker Points** | Estimativa de esforço para tarefas (planning poker) |
| **Flag Pirata** | Marcação de tarefa não planejada que entrou na sprint |
| **targetAudience** | Campo que define para quais contratos uma pergunta aparece (CLUBE, CORP, START, AMBOS) |
| **selectedPillarId** | UUID do pilar escolhido por escolas Start (restringe acesso) |

## Técnico

| Termo | Significado |
|-------|------------|
| **NSSM** | Non-Sucking Service Manager — gerenciador de serviços Windows usado em produção |
| **MediaMTX** | Servidor de streaming open-source que converte RTMP para HLS |
| **HLS** | HTTP Live Streaming — protocolo de streaming de vídeo |
| **RTMP** | Real-Time Messaging Protocol — protocolo usado pelo OBS para transmitir |
| **Flyway** | Ferramenta de migração de banco de dados SQL |
| **EntityHelper** | Utilitário obrigatório do backend para lookups de entidades |
| **Toggleable** | Interface obrigatória para entidades com campo `active` |
| **useAdminResource** | Hook genérico do frontend para CRUD admin |
